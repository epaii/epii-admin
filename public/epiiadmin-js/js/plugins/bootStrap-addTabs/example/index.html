Tab add close drop for Bootstrap bootstrap.addtabs.js

About

bootstrap.addtabs.js是一个bootstrap的TAB扩展插件。可以使用它自动创建、关闭、折叠、拖动tabs，创建的tabs可以使用ajax/iframe加载，或者直接指定内容。同时，还可以在tabs标签上使用右键关闭、刷新tab。
Bootstrap.addtabs.js is a bootstrap of the tabs extension. You can use it to automatically create, close, drop, drag tabs, create tabs can be used to load ajax / iframe, or directly specify the content. At the same time, you can also use the right tab on the tabs tab to close, refresh the tab.

I'm a homepage.
一般使用方法 Basic
HTML:
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt">mailbox</button>
JS: 
$('#tabs-1').addtabs();
使用Ajax Use Ajax
HTML: 
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt" data-ajax="true">
    mailbox
</button>
JS: 
$('#tabs-1').addtabs();
不允许关闭 Not allowed to close
HTML:
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt">mailbox</button>
JS: 
$('#tabs-1').addtabs({'close':false});
指定标题 Customize the title
HTML: 
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt" data-title="Customize the title">
    ...
</button>
JS: 
$('#tabs-1').addtabs();
指定内容 Customize the content
HTML: 
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt" data-content="Customize the content">
    ...
</button>
JS: 
$('#tabs-1').addtabs();
Note: 
如果设置了指定内容，data-url属性将不起作用。
不使用右键菜单 Do not use the contextmenu
HTML:
<button class="btn btn-default" data-addtab="mail" data-url="./ajax/mailbox.txt">mailbox</button>
JS: 
$('#tabs-1').addtabs({'contextmenu':false});

contextmenu bool true 是否启用右键菜单
callback function(){} 关闭后回调函数
local
